# Raspberry Pi Web Camera

Raspberry Pi Web Camera is web camera build on Raspberry Pi 2 which uses Raspberry Pi Camera.

  - Interface via web page
  - Motion detection
  - Taking photos and recordings

### Version
1.0.0

### List of third party libraries
- [Picamera]
- [NumPy]
- [Django]
- [Gevent]
- [Simplex]
